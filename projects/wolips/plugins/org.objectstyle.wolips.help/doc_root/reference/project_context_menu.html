Project Context Menu

You access this by right-clicking on the project folder in the Navigator or Package Explorer views (as well as several other views).

WOLips specific

  • Update PB.project - Refreshes or repairs the PB.project file. Normally this shouldn't be necessary. If it, and this doesn't work, you can try Close Project and then Open Project.
  • Add Local Framework - Lets you add a WO framework located under NEXT_LOCAL_ROOT to the project's Java Build Path.
  • Add System Framework - Lets you add a WO framework located under NEXT_SYSTEM_ROOT to the project's Java Build Path.
  • Remove Target Builder - See the TargetBuilder section.
  • Update Target Builder - See the TargetBuilder section.

General (selection)

  • New - Run one of the wizards.
  • Close Project - ?
  • Properties - Access project properties.
